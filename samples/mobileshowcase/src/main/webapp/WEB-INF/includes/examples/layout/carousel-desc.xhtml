The carousel component displays a list of child elements in a circular fashion allowing a user to swipe right or left to view non visible child elements. The most common use of this component is for displaying a list of image where the user can swipe to view more images in the list.

The component tag lives in the mobic name space as is it is currently defined as a Facelets composite component. Future release will introduced this component as a concrete JSF component. 

<mobic:carousel cols="1" >
    <mobic:carouselItem>
        carousel cell content
    </mobic:carouseltItem>
</mobic:carousel>

The carousel can be moved with either a mouse drag or as a touch swipe gesture.
