The qrscan and qrcode components can be used to read or create QR codes. The ICEmobile container is needed for reading QR codes.

The scrscan component will access the devices camera and try and read a QR code. If successful the decode qr code will be assigned to the backing bean.

<mobi:qrscan
    value="#{'#'}{myBean.qrcode}"/>

The srcode component can be used to write QR code graphics. The component takes a string value and will return the corresponding QR code graphic.

<mobi:qrcode
    value="#{'#'}{myBean.encode}"/>
