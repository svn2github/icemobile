The mobi:commandButton extends h:commandButton adding named types and device theme support as well as a selected state for combination with a button Group.

    <mobi:buttonGroup
            orientation="horizontal">
        <mobi:commandButton
                selected="false">
            Yes</mobi:commandButton>
        <mobi:commandButton
                selected="true">
            No</mobi:commandButton>
    </mobi:buttonGroup>

The component tag lives in the mobi name space as is it is currently defined as a Facelets composite component. Future release will introduced this component as a concrete JSF component.
