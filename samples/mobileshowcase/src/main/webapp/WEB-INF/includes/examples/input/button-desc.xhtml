The mobic:commandButton extends h:commandButton adding named types and device theme support as well as a selected state for combination with a button Group.

    <mobic:buttonGroup
            orientation="horizontal">
        <mobic:commandButton
                selected="false">
            Yes</mobic:commandButton>
        <mobic:commandButton
                selected="true">
            No</mobic:commandButton>
    </mobic:buttonGroup>

The component tag lives in the mobic name space as is it is currently defined as a Facelets composite component. Future release will introduced this component as a concrete JSF component.
