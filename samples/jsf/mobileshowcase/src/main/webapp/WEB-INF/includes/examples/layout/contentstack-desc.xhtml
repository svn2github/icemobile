The contentStack component can be used to display a stack of contentPane components, one of which will be visible at a time. The contentStack can be used for single page navigation, as well as for controlling page fragments like wizards. The contentStack specifies the id of the child contentPane that is visible. 

   <mobi:contentStack
       contentId="panel1" >
      <mobi:contentPane
        id="pane1" >
         ...
      </mobi:contentPane>
      <mobi:contentPane
        id="pane2" >
         ...
      </mobi:contentPane>
   </mobi:contentStack>
