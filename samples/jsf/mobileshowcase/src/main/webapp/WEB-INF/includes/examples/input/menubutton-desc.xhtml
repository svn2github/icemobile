The menuButton allows a user to select and execute a menu item. ActionListener methods can be assigned for each item in the menu. The menuButton component can also be combined with panelConfirmation and submitNotification.

<mobi:menuButton
        buttonLabel="Action"
        value="#{bean.menu}"
                 var="item">
  <mobi:menuButtonItem
    label="#{item.label}"
    actionListener="#{item.execute}"/>
</mobi:menuButton>
