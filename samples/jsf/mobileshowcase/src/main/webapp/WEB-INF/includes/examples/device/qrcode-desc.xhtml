The scan and qrcode components are used to read or create QR codes. The ICEmobile Container is needed for reading QR codes.

The scan component accesses the device camera to read a QR code. If successful, the decoded QR Code will be assigned to the specified backing bean property.

The button label can be changed after a code has been scanned to notify the user. The 'buttonLabel' attribute is first displayed, and after a code has been scanned, the 'captureMessageLabel' attribute value is then shown as the button label. 

<mobi:scan id="scanOne"
  value="#{bean.qrScanner}"
  buttonLabel="Scan QR Code"
  captureMessageLabel="Captured"/>

The qrcode component is used to generate QR code graphics. The component takes a string value and returns the corresponding QR code graphic.

 
<mobi:qrcode
    value="#{myBean.encode}"/>
