Meet BridgeIt!
The easiest way to add native features to your web application.

Use a simple JavaScript API to access native features, such as the camera, camcorder, contact list, microphone, scanner, and even augmented reality!

Go beyond HTML5...

  • Scanner

Camera Access

Access the native device camera features with a simple line of JavaScript...

ice.mobi.camera(id,callback,   {postURL:myURL});
Take a Photo!

Access the native video recording features with a simple line of JavaScript...

ice.mobi.camcorder(id,callback,   {postURL:myURL});
Record some video!
